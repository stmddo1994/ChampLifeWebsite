Images

The <img> tag embeds images in web pages. Always include:

  • src - Path to the image file
  • alt - Alternative text for accessibility

Example:

<img src="champlifelogo.png" alt="A beautiful landscape">

Audio

Embed audio with the <audio> tag:

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
Your browser doesn't support HTML5 audio.
</audio>

Example audio player:

Video

Embed video with the <video> tag:

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
Your browser doesn't support HTML5 video.
</video>

Responsive Media

Use the <picture> element for responsive images:

<picture>
  <source media="(min-width: 800px)" srcset="large.jpg">
  <img src="small.jpg" alt="Responsive image">
</picture>
